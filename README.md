### Open the directory and run the script line:

#### Install Packages

```bash
yarn
```

#### Testing

Create a test.env file in the root directory and include the following entries:

```bash
PRIVATE_KEY=""
PUBLIC_KEY=""
FROM_ADDRESS=""
TO_ADDRESS=""
WALLET_ADDRESS=""
```

Then run:

```bash
yarn test
```

#### build node

```bash
yarn build:node
```

#### build browser

```bash
yarn build:browser
```
